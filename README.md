## Описание проекта

Проект представляет собой решение хакатона Hack-n-Change-2025-MTC-Web-DA, для автоматизированного сбора и анализа статистики контента с использованием платформы MWS Tables. Решение включает три основных компонента:

1. **Интеграционная схема данных** - автоматический workflow в MWS Tables для сбора статистики с YouTube канала, обработки данных и загрузки результатов в таблицу результатов. Схема использует API YouTube для получения информации о видео, включая количество просмотров, лайков, комментариев и других метрик.

2. **Backend сервис** - Python приложение, которое взаимодействует с YouTube API, собирает статистику по видео в реальном времени и готовит данные в формате, совместимом с MWS Tables.

3. **AI чат виджет** - веб-интерфейс на основе OpenRouter API, позволяющий пользователям задавать вопросы о контенте на естественном языке и получать ответы с использованием языковых моделей.

Основная цель проекта - создать интегрированное решение для управления контентом, которое объединяет сбор данных, их обработку и анализ в едином пространстве.
